# React Redux Webpack Todo Demo App

## Todo app demo

## Features

-   react 16
-   react-router
-   babel 7
-   webpack 4
-   jest/enzyme
-   sass/scss
-   eslint/prettier
-   husky (pre-commit hooks)

## To run app

In a terminal window run `npm run dev-server`

In a 2nd terminal window run `npm start`
